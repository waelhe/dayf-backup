## 5. متغيرات البيئة

### Dayf (.env)

```env
# Supabase
NEXT_PUBLIC_SUPABASE_URL=https://xxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=xxx
SUPABASE_SERVICE_ROLE_KEY=xxx

# App
NEXT_PUBLIC_APP_URL=https://dayf.app
```

### Discourse (.env)

```env
# Database
DISCOURSE_DB_HOST=localhost
DISCOURSE_DB_PORT=5432
DISCOURSE_DB_NAME=discourse
DISCOURSE_DB_USERNAME=discourse
DISCOURSE_DB_PASSWORD=xxx

# Redis
DISCOURSE_REDIS_HOST=localhost
DISCOURSE_REDIS_PORT=6379
DISCOURSE_REDIS_DB=1

# App
DISCOURSE_HOSTNAME=community.dayf.app
DISCOURSE_DEVELOPER_EMAILS=admin@dayf.app

# SSO
DISCOURSE_SSO_URL=https://dayf.app/api/sso
DISCOURSE_SSO_SECRET=xxx
```

### Saleor (.env)

```env
# Database
DATABASE_URL=postgresql://saleor:xxx@localhost:5432/saleor

# Redis
REDIS_URL=redis://localhost:6379/2

# Celery
CELERY_BROKER_URL=redis://localhost:6379/2
CELERY_RESULT_BACKEND=redis://localhost:6379/2

# App
ALLOWED_HOSTS=api.dayf.market,market.dayf.app
SECRET_KEY=xxx

# SSO
EXTERNAL_AUTH_URL=https://dayf.app/api/auth/external
```
